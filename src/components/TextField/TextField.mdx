# TextField

A labelled text input built on [react-aria-components `TextField`](https://react-spectrum.adobe.com/react-aria/TextField.html). Supports helper text, error messages, required marking, disabled, and read-only states. Fully keyboard and screen reader accessible.

## Usage

```tsx
import TextField from '@/components/TextField';

<TextField
  label="Email address"
  placeholder="you@example.com"
  helperText="We'll never share your email."
/>
```
