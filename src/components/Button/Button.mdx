# Button

A general-purpose button component built on top of [react-aria-components](https://react-spectrum.adobe.com/react-aria/Button.html) for full keyboard and screen reader accessibility. Styled with Tailwind CSS.

## Usage

```tsx
import Button from '@/components/Button';

<Button variant="primary" size="md">Click me</Button>
```

## Variants

Use `variant` to communicate the intent of the action.

- **primary** — the main call-to-action on a page
- **secondary** — a supporting or alternative action
- **destructive** — irreversible actions like deleting or clearing data
