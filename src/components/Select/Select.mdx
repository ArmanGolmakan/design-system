# Select

A dropdown select built on [react-aria-components `Select`](https://react-spectrum.adobe.com/react-aria/Select.html). The listbox opens in a `Popover` aligned to the trigger width. Supports helper text, error messages, required marking, and disabled options. Fully keyboard and screen reader accessible.

## Usage

```tsx
import Select from '@/components/Select';

const items = [
  { id: 'apple', label: 'Apple' },
  { id: 'banana', label: 'Banana' },
];

<Select
  label="Favourite fruit"
  placeholder="Select a fruit"
  items={items}
/>
```

For custom option rendering, import `SelectOption`:

```tsx
import Select, { SelectOption } from '@/components/Select';
import type { SelectItem } from '@/components/Select';

<Select label="Fruit" items={items}>
  {(item) => <SelectOption id={item.id}>{item.label}</SelectOption>}
</Select>
```
