# Checkbox

A single checkbox built on [react-aria-components `Checkbox`](https://react-spectrum.adobe.com/react-aria/Checkbox.html). Supports checked and disabled states. Fully keyboard and screen reader accessible.

## Usage

```tsx
import Checkbox from '@/components/Checkbox';

<Checkbox defaultSelected>Accept terms and conditions</Checkbox>
```
