# Modal

A dialog overlay built on [react-aria-components `Modal` and `Dialog`](https://react-spectrum.adobe.com/react-aria/Modal.html). Traps focus, closes on Escape, and announces correctly to screen readers. Supports both `dialog` and `alertdialog` roles.

## Usage

```tsx
import Modal, { ModalTrigger } from '@/components/Modal';
import Button from '@/components/Button';

<ModalTrigger>
  <Button>Open modal</Button>
  <Modal
    title="Edit profile"
    footer={
      <>
        <Button variant="secondary">Cancel</Button>
        <Button>Save changes</Button>
      </>
    }
  >
    <p>Make changes to your profile here.</p>
  </Modal>
</ModalTrigger>
```
